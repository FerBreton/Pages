    Objetivos

    Objetivo general


    Conocer, identificar y comprobar el funcionamiento de una página web con programación PHP, en un simulador, un servidor local y envío de datos mediante el explorador.


    Objetivos específicos


    - Utilizando Replit desarrollar una página web con código PHP que permita actualizar el valor de un sensor. Recuerda guardar el archivo como "index.php", corrobora su funcionamiento directo desde el explorador. Utilizando tu página web, inserta el desarrollo en Replit como código embebido en tu reporte de práctica en tu página web y explica los comandos.


    - Finalmente utilizando tu servidor local con xampp, desarrolla una página PHP de visualización para 4 sensores ue se encuentre directamente en tu página web, visualízala en el explorador con la dirección IP y corrobora su funcionamiento directo desde el explorador desde otros 2 dispositivos de la red local.
